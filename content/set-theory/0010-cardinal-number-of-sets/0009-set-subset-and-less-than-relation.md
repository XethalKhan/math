# Lemma - set subset and less than relation

If $A$ and $B$ are sets with $A \subseteq B$, then $A \prec B$

## Proof

Clearly the inclusion mapping $i_A: A \rightarrow B$ is a one-to-one mapping of $A$ into $B$.
