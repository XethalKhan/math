# Basic properties of relation inverse

Let $R$ be a nonempty relation from a set $A$ to a set $B$. Then

1. $dom(R^{-1}) = rng(R)$
2. $rng(R^{-1}) = dom(R)$
3. $R = (R^{-1})^{-1}$
4. $R = R^{-1}$ if and only if $A = B$ and $R$ is **symmetric**
5. $R \cap R^{-1} = I_A$ if and only if $A = B$ and $R$ is **antisymmetric**
