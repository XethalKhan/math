## De Morgan's law

For all sets $X$, $Y$ and $Z$ in any universal set $U$

1. $(X \cap Y)' = X' \cup Y'$
2. $(X \cup Y)' = X' \cap Y'$
3. $X - (Y \cup Z) = (X - Y) \cap (X - Z)$
4. $X - (Y \cap Z) = (X - Y) \cup (X - Z)$
