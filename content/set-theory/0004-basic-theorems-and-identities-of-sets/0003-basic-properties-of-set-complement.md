# Basic properties of set complement

For set $X$ in any universal set $U$

1. $X'' = X$ (law of double complement)
2. $X \cup X' = U$
3. $X \cap X' = \emptyset$
4. $U' = \emptyset$
5. $\emptyset' = U$
