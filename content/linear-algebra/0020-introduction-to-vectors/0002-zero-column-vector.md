# Zero column vector

## Definition

The **zero vector** of size $m$ is the **column vector** of size $m$ where each entry is the number zero

$$
0 =
\begin{bmatrix}
    0\\
    0\\
    \vdots\\
    0
\end{bmatrix}
$$

or defined much more compactly $0_i = 0$ for $1 \leq i \leq m$
