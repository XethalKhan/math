# Matrix addition

## Definition

Given the $m \times n$ matrices $A$ and $B$, define the sum of $A$ and $B$ as an $m \times n$ matrix, written $A + B$, according to

$$
[A + B]_{ij} = [A]_{ij} + [B]_{ij}, 1 \leq i \leq m, 1 \leq j \leq n
$$
